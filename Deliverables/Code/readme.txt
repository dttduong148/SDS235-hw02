Movement Heatmap.py
	This file requires pandas and plotly.express libraries. The data used for the file are park_movement datasets for all three days. Please place the file and datasets in the same folder to check.

Communication numbers over time by locations.ipynb
	Put data and the code in the same file to see the result. This one shows numbers of communication over time by locations. There are three visualizations for Friday, Saturday and Sunday.

movement_tracking.qmd
	Used coordinate record from weekend data to track movements of all visitors and suspicious individuals on Friday, Saturday, Sunday. Included an animation of movement on Saturday.
